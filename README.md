# 台灣即時天氣查詢系統

## 專案介紹

本專案使用 HTML、CSS 與 JavaScript 製作前端介面，透過 Fetch API 呼叫天氣 API 取得即時天氣資料，讓使用者能快速查詢台灣各城市的天氣資訊。

系統會顯示城市天氣資訊，例如：

- 溫度
- 濕度
- 天氣狀況

本專案同時結合 Google Apps Script（GAS）與 Google Sheet，自動儲存使用者查詢紀錄，最後透過 GitHub Pages 發布網站。

## 使用技術

- HTML
- CSS
- JavaScript
- Fetch API
- Google Apps Script（GAS）
- Google Sheet
- GitHub Pages

## 功能介紹

1. 輸入城市名稱查詢天氣
2. 顯示即時溫度
3. 顯示空氣濕度
4. 顯示天氣狀況
5. 自動儲存查詢資料

## 系統流程

使用者輸入城市

↓

JavaScript Fetch API

↓

天氣 API

↓

Google Apps Script

↓

Google Sheet
